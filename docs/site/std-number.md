# `number`

Primitive number methods are exposed through `Number`.

Current methods:

- `abs()`
- `clamp(min, max)`
- `floor()`
- `ceil()`
- `round()`
- `i()`
- `f()`
- `s(format = 'decimal')`
- `zero()`
- `positive()`
- `negative()`
- `between(min, max)`
- `incr(step = 1)`
- `decr(step = 1)`
- `times(&blk)`
