# Roadmap

Jaya is already usable for experimentation, but the language and runtime are still moving.

Areas expected to grow:

- more stdlib depth, especially `fs`, `path`, `http`, and `sql`
- richer math support, including larger numeric and algebra tooling
- more polished docs and examples
- better packaging, potentially with an embedded runtime

Current documentation aims to describe what already exists in the implementation, and to clearly separate that from planned work.
