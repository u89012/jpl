# `string`

Primitive string methods are exposed through `String`.

Current methods:

- `trim()`
- `upcase()`
- `downcase()`
- `size()`
- `empty()`
- `includes(needle)`
- `startsWith(prefix)`
- `endsWith(suffix)`
- `snakeCase()`
- `camelCase()`
- `pascalCase()`
